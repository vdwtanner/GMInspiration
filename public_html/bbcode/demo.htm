BBCode generator PHP HTML decoder script demo

This demo script will give you a realistic idea that how successfully this JavaScript editor can generate BBCode for you. A preview system is also included here. To generate this preview I use AJAX technology to submit BBCode text to the web server. The PHP script at the server end contains a series of functions which decode the BBCode text to its HTML equivalent.

For preview purpose the BBCode text is limited to 300 characters.

If you have any question or doubt about use this script please do not heisted to contact me.
